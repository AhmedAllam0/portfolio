شرح الملفات الرئيسية

index.html

الملف الأساسي الذي يحتوي على جميع أقسام الموقع: Hero, About, Skills, Portfolio, Blog, Services, Pricing, FAQ, Testimonials, Contact.

style.css

يحتوي على جميع التنسيقات والألوان والحركات. يمكنك تعديل الألوان والخطوط من هنا بسهولة.

script.js

يدير جميع التفاعلات: نظام الثيمات، الصوتيات، المؤشر الحي، النماذج، والتأثيرات المتحركة.

دليل التخصيص

1. تخصيص الألوان

لتغيير الألوان الرئيسية، افتح ملف style.css وابحث عن المتغيرات في بداية الملف:

:root {
    --primary-rgb: 102, 126, 234;      /* اللون الأساسي */
    --secondary-rgb: 237, 100, 166;    /* اللون الثانوي */
    --accent-rgb: 255, 159, 64;        /* لون التمييز */
}

/* مثال: لتغيير اللون الأساسي للأخضر */
:root {
    --primary-rgb: 34, 197, 94;        /* أخضر */
}

2. تغيير الصور

  1. احفظ صورك في مجلد assets/images/
  2. افتح index.html وابحث عن العنصر <img> الذي تريد تغييره
  3. غيّر قيمة src="" لتشير لصورتك الجديدة
<!-- قبل -->
<img src="https://example.com/old-image.jpg" alt="صورة">

<!-- بعد -->
<img src="assets/images/my-photo.jpg" alt="صورة">

3. تعديل المحتوى النصي

جميع النصوص موجودة في index.html، ابحث عن القسم الذي تريد تعديله:

قسم Hero (الرئيسية)

ابحث عن <section id="home">

قسم About (عني)

ابحث عن <section id="about">

قسم Skills (المهارات)

ابحث عن <section id="skills">

4. إضافة/تعديل مشاريع Portfolio

لإضافة مشروع جديد، انسخ البنية التالية والصقها داخل قسم Portfolio:

<div class="portfolio-card">
    <a href="رابط_المشروع" class="glightbox">
        <img src="صورة_المشروع.jpg" alt="اسم المشروع">
        <div class="portfolio-card-overlay">
            <h3>اسم المشروع</h3>
            <p>وصف قصير للمشروع</p>
            <div class="portfolio-card-tags">
                <span>HTML</span>
                <span>CSS</span>
            </div>
        </div>
    </a>
</div>

نشر الموقع

GitHub Pages (مجاني)

  1. أنشئ حساب على GitHub إن لم يكن لديك
  2. أنشئ Repository جديد باسم username.github.io
  3. ارفع ملفات المشروع للـ Repository
  4. افتح Settings → Pages → اختر Branch: main
  5. موقعك الآن متاح على https://username.github.io

Netlify (مجاني)

  1. انتقل إلى netlify.com وأنشئ حساب
  2. اسحب مجلد المشروع وأفلته في صفحة Netlify
  3. انتظر دقيقة واحدة للنشر التلقائي
  4. احصل على رابط مجاني مثل yoursite.netlify.app

Vercel (مجاني)

  1. انتقل إلى vercel.com وسجل دخول بـ GitHub
  2. اضغط "New Project" واختر Repository الخاص بك
  3. اضغط "Deploy" وانتظر دقائق قليلة
  4. موقعك جاهز على yoursite.vercel.app

الأسئلة الشائعة

هل أحتاج خبرة برمجية لاستخدام القالب؟

لا، القالب سهل الاستخدام للمبتدئين. كل ما تحتاجه هو تعديل النصوص والصور في ملف HTML. لكن معرفة أساسيات HTML/CSS ستساعدك في التخصيص المتقدم.

هل القالب متوافق مع الموبايل؟

نعم بالتأكيد! القالب مصمم بتقنية Responsive Design ويعمل بشكل مثالي على جميع الأجهزة (موبايل، تابلت، كمبيوتر).

هل يمكنني استخدام القالب لمشاريع تجارية؟

نعم، يمكنك استخدام القالب لأي غرض تجاري أو شخصي بدون قيود.

كيف أضيف نموذج اتصال يعمل فعلياً؟

يمكنك استخدام خدمات مثل EmailJS, Formspree, أو Google Forms. راجع قسم "تكامل نموذج الاتصال" في التوثيق.

هل هناك دعم فني متاح؟

نعم، إذا واجهت أي مشكلة يمكنك التواصل معنا عبر البريد الإلكتروني أو فتح Issue على GitHub.
